# Data

Place the dataset files in this directory.

Download the Kaggle "Brain Tumor for 14 classes" dataset from:
https://www.kaggle.com/datasets/waseemnagahhenes/brain-tumor-for-14-classes

Extract it under `data/brain_tumor_14_classes/` and follow the preparation
instructions in the root `README.md` (Section 5.2) to filter, deduplicate, and
split the data before training.
